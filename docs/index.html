Psi_k frees you from the login shell

Jobs and files are the bread and butter of computing.

Get them right, and new possibilities open up for getting your work done. Get them wrong and you're in printer hell.

Open-source BSD-3-Clause License.

Callback-Driven

Ditch polling. Register callback actions to fire automatically when jobs change state.

Embrace REST

API route names follow REST principles. Jobs have structure, and developers are happy.

Full Authn+Authz

Never enter a password. Client certificates authenticate each connection, and Biscuit tokens grant fine-grained, user-attenuated, authorizations.

TLSv1.3

Latest web transport security, with ed25519 public-key cryptography

Configured Queues

Don't add useless parameters to jobs. Queue types select common configurations. Add new backends to drive your existing interfaces.

Pydantic Serialization

Strict typing for all inputs, outputs, and internal functions removes the ambiguities that can lead to hacks.
